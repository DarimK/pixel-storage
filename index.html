Pixel Storage

This is a simple web tool for embedding and extracting data within the pixels of an image. This is an established practice known as Steganography.

To embed data in an image, the least significant bits of each color channel (R, G, B) are replaced with bits from the data. To extract data, these same bits are read from the image to reconstruct the original data. The number of bits per color channel you choose affects the amount of data that can be stored. A value of 1 provides low storage capacity but results in a nearly identical image, while a value of 8 provides high storage capacity but completely replaces the old image colors.
